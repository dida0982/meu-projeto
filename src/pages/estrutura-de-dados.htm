Introdução a Estrutura de Dados

-Stack = O último a chegar é o primeiro a sair.
*ponto negativo; ruim para filas de espera.

-Queue = primeiro a chegar é o primeiro a sair.
*ponto negativo; entradas urgentes precisam arperar.

Priority Queue = Os casos mais urgentes primeiro.
Sistema mais lento e complexo.

Estrura de dados = Organiza e armazena dados de forma eficiente.
*Se feita de forma errada = trava o site, causa riscos a segurança, ...

Algorithms = É um conjunto de instruções bem definidas. Para resolver problems e exacutar tarefas.
